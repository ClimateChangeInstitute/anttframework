What is available

Access AntT data base.

Geochemistry

Complied comma separated files (CSV format) for each region.

PDF files

Download pdf file with total alkali vs. silica (TAS diagram after Le Bas et al., 1986). For all plots corresponding data available as CSV files.

R scripts

Download R script file that was used to generate pdf file with all data from the region. Open downloaded R script in R free software environment for statistical computing and graphics and generate you own PDF plots using geochemistry CSV files hosted on Antt server.
